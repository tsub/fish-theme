# fish-theme

My fish theme

![fish-theme]

## Requirements

* [Nerd Font](https://github.com/ryanoasis/nerd-fonts)
* [Ladicle/git-prompt](https://github.com/Ladicle/git-prompt)
* [kubectl](https://kubernetes.io/docs/reference/kubectl/overview/)

## Install

With [fisher]

```
fisher install tsub/fish-theme
```
